# Quickzii Rider Support Agent

**MSc AI Dissertation** by Sadiq (BSBI / UCA Berlin)
**Supervisor:** Professor Vincent English

Human-in-the-loop AI conversational agent for automating rider support in logistics operations.

## Repository structure

- `mock_motiontools/` — FastAPI service simulating the MotionTools dispatch backend
- `rider_support_agent/` — Claude-powered agent with tool-use, orchestration, escalation, and audit logging
- `quickzii_app/` — Next.js frontend (Rider Chat, Bookings, Drivers, Human Agent Dashboard)
